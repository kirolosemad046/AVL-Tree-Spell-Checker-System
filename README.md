# AVL Tree Spell Checker System
# Description
A simple spell checker system built using AVL Tree in C.  
It loads words from a dictionary file and checks spelling in input sentences.
##  Features
- AVL Tree implementation (self-balancing BST)
- Insert and search words efficiently
- Spell checking for sentences
- Suggestions using nearby words (predecessor & successor)
- File handling for dictionary loading
## 🧠 Concepts Used
- AVL Tree
- Binary Search Tree
- Recursion
- File Handling
- String manipulation
